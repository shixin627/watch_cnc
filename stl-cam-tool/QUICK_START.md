# 快速啟動指南

## 立即使用

### 1. 啟動應用程式

```bash
cd stl-cam-tool
npm start
```

或者在開發模式下啟動(會開啟開發者工具):

```bash
npm run dev
```

### 2. 使用流程

#### 步驟 1: 載入 STL 檔案
- 點擊左上角「載入 STL 檔案」按鈕
- 選擇您的 .stl 檔案
- 模型將顯示在 3D 視窗中

#### 步驟 2: 選擇加工區域
- 點擊「啟用選擇模式」
- 在模型上拖曳滑鼠,創建綠色選擇框
- 點擊「停用選擇模式」完成選擇

#### 步驟 3: 調整參數
在右側面板設定:
- 刀具直徑: 1.0 mm (預設)
- 步距比例: 40% (預設)
- 每層深度: 0.5 mm (預設)
- 進給速度: 100 mm/min (預設)

#### 步驟 4: 生成路徑
- 點擊「生成刀具路徑」
- 等待生成完成
- 紅色線條會顯示生成的路徑

#### 步驟 5: 匯出 G-code
- 點擊「匯出 G-code」
- 選擇儲存位置
- 完成!您的 .nc 檔案已就緒

## 視窗操作

- **旋轉視角**: 左鍵拖曳
- **平移視角**: 右鍵拖曳
- **縮放**: 滾輪
- **重置**: 點擊「重置視角」

## 注意事項

1. **選擇區域太小**: 確保選擇框有足夠大的體積
2. **路徑生成緩慢**: 降低模型解析度或減小選擇區域
3. **記憶體不足**: 處理大型 STL 時可能需要較多記憶體

## 打包成執行檔

如果您想要獨立的 .exe 檔案:

```bash
npm run build:win
```

執行檔會在 `dist/` 目錄中生成。

## 問題排除

### 無法啟動
- 確認已安裝 Node.js (v16+)
- 執行 `npm install` 確保依賴已安裝

### 無法載入 STL
- 確認檔案格式為 .stl
- 支援 Binary 和 ASCII 格式

### 路徑生成失敗
- 檢查是否已選擇區域
- 確認選擇區域與模型有交集
